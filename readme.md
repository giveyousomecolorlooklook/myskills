# 我的 Codex 用户技能

这个仓库用于保存和维护本机 Codex 的用户技能副本。每个 `user-*` 目录都是一个独立技能，核心说明在对应目录的 `SKILL.md`。

## 技能列表

### user-blp1-converter

Warcraft III `BLP1` 贴图和 `PNG` 之间的转换工具。

### user-imagegen

基于脚本的图片生成/编辑技能,使用 gpt-image-2,支持第三方baseurl和key


### user-ui-preview-atlas

把一张 UI 预览图拆解为组件汇总图，尤其适合游戏 UI mockup 或绿幕背景 UI 预览图。该技能会配合 `user-imagegen` 使用，先分析组件，再生成可切片的 atlas。


### user-green-screen-cutout

用于去除纯色绿幕/色键背景，输出带透明通道的 PNG。
